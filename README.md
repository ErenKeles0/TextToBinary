# 🔡 TextToBinary  

Convert text to binary and binary back to text — simple, fast, and easy to use.  

---

## ✨ Features
- 📝 Convert **text → binary**
- 🔄 Convert **binary → text**
- ⚡ Fast, lightweight, and beginner-friendly
- 💻 Written in **C++**
